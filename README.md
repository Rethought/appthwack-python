appthwack-python
================

The official [AppThwack](https://appthwack.com) python client.

Status
======

Currently under active development.

Installation
============

### Source

    $ git clone git@github.com:appthwack/appthwack-python.git
    $ python setup.py install

### Pip

    $ pip install appthwack
